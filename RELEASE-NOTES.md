# Ping Tester 1.0.0

The first public release of Ping Tester, a portable desktop application for measuring connection stability on Windows without working directly in a terminal.

## Included

- Tests configurable by duration and interval against multiple IP addresses or hostnames.
- Real-time latency, packet loss, and outage monitoring.
- Charts, result summaries, and individual sample details.
- Automatic CSV and JSON exports with a local run history.
- External result file viewing without importing or modifying the source files.
- English and Spanish interfaces.
- A single portable Windows x64 executable with no .NET installation required.

## Requirements

- 64-bit Windows 10 or Windows 11.
- Windows PowerShell 5.1.
- Microsoft Edge WebView2 Runtime.

Download the attached `PingTester-v1.0.0-win-x64.exe` file and run it directly.
